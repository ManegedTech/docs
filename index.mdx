## What is Maneged?

Maneged is a modern team collaboration platform that combines project management, real-time messaging, calendar scheduling, and task tracking in one unified solution. Built with Flutter for mobile and Node.js/Express with WebSocket for the backend, Maneged provides seamless real-time communication for teams of all sizes.

## Key Features

### 💬 Real-time Messaging
- Individual and group chat with instant delivery
- Message reactions and threading
- Typing indicators and read receipts
- File attachments (images, documents, videos)

### 📊 Project Management
- Task creation and assignment
- Project tracking and progress monitoring
- Team-based organization
- Role-based access control

### 📅 Calendar & Scheduling
- Event creation and management
- Team scheduling coordination
- Reminder notifications
- Calendar integration with tasks

### 🔔 Smart Notifications
- Push notifications via Firebase Cloud Messaging
- Real-time presence system
- Multi-device support
- Customizable notification preferences

### 🔒 Enterprise Security
- API key management with fine-grained scopes
- JWT-based authentication
- Organization-level data isolation
- Role-based permissions

## Architecture

Maneged uses a modern, scalable architecture:

```
┌─────────────────┐         ┌─────────────────┐
│  Flutter Mobile │◄───────►│   Node.js API   │
│      App        │  REST   │   + WebSocket   │
└─────────────────┘         └─────────────────┘
        │                            │
        │                            │
        ▼                            ▼
┌─────────────────┐         ┌─────────────────┐
│  Isar (Local)   │         │    Appwrite     │

        │                            │
```
