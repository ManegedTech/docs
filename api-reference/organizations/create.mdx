---
title: 'Create Organization'
api: 'POST /v1/organization/create'
description: 'Create a new organization'
---

## Endpoint

```
POST /v1/organization/create
```

## Authentication

Requires user token or API key with `write:organizations` scope.

## Request Body

<ParamField body="name" type="string" required>
  Organization name
</ParamField>

<ParamField body="description" type="string">
  Organization description
</ParamField>

<ParamField body="is_public" type="boolean">
  Whether organization is publicly searchable (default: false)
</ParamField>

## Example Request

<CodeGroup>
```bash cURL
curl -X POST \
  -H "x-api-key: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Company",
    "description": "A great company",
    "is_public": true
  }' \
  https://api.maneged.com/v1/organization/create
```

```javascript JavaScript
const response = await fetch('https://api.maneged.com/v1/organization/create', {
  method: 'POST',
  headers: {
    'x-api-key': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'My Company',
    description: 'A great company',
    is_public: true
  })
});
```
</CodeGroup>

## Response

```json
{
  "status": 201,
  "success": true,
  "data": {
    "$id": "org-123",
    "name": "My Company",
    "description": "A great company",
    "code": "ABC123",
    "is_public": true,
    "owner_id": "user-123",
    "created_at": "2025-01-19T10:00:00.000Z"
  }
}
```

<Note>
  A unique join code is automatically generated for the organization.
</Note>
