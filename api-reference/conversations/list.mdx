---
title: 'List Conversations'
api: 'GET /v1/conversation'
description: 'Get all conversations for authenticated user'
---

## Endpoint

```
GET /v1/conversation
```

## Authentication

Requires user token or API key with `read:conversations` scope.

## Query Parameters

<ParamField query="organizationId" type="string">
  Filter by organization ID (optional)
</ParamField>

<ParamField query="unreceived" type="boolean">
  Get only unreceived conversations (default: false)
</ParamField>

<ParamField query="limit" type="number">
  Number of results (default: 25, max: 100)
</ParamField>

<ParamField query="offset" type="number">
  Pagination offset (default: 0)
</ParamField>

## Example Request

<CodeGroup>
```bash cURL
curl -X GET \
  -H "x-api-key: Bearer YOUR_TOKEN" \
  "https://api.maneged.com/v1/conversation?limit=10"
```

```javascript JavaScript
const response = await fetch('https://api.maneged.com/v1/conversation?limit=10', {
  headers: {
    'x-api-key': 'Bearer YOUR_TOKEN'
  }
});
```
</CodeGroup>
