---
title: 'Update User Profile'
api: 'POST /v1/user/profile'
description: 'Update user profile information'
---

## Endpoint

```
POST /v1/user/profile
```

## Authentication

Requires user token or API key with `write:users` scope.

## Request Body

<ParamField body="name" type="string">
  User's full name
</ParamField>

<ParamField body="avatar_url" type="string">
  Profile picture URL
</ParamField>

<ParamField body="status" type="string">
  User status (online, offline, busy)
</ParamField>

## Response

<ResponseField name="status" type="number">
  HTTP status code
</ResponseField>

<ResponseField name="success" type="boolean">
  Whether the update was successful
</ResponseField>

<ResponseField name="data" type="object">
  Updated user profile
</ResponseField>

## Example Request

<CodeGroup>
```bash cURL
curl -X POST \
  -H "x-api-key: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name": "Jane Doe", "status": "busy"}' \
  https://api.maneged.com/v1/user/profile
```

```javascript JavaScript
const response = await fetch('https://api.maneged.com/v1/user/profile', {
  method: 'POST',
  headers: {
    'x-api-key': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Jane Doe',
    status: 'busy'
  })
});
```
</CodeGroup>

## Example Response

```json
{
  "status": 200,
  "success": true,
  "data": {
    "$id": "user-123",
    "name": "Jane Doe",
    "status": "busy",
    "updated_at": "2025-01-19T10:05:00.000Z"
  }
}
```
