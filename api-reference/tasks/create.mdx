---
title: 'Create Task'
api: 'POST /v1/tasks/create'
description: 'Create a new task'
---

## Endpoint

```
POST /v1/tasks/create
```

## Authentication

Requires user token or API key with `write:tasks` scope.

## Request Body

<ParamField body="title" type="string" required>
  Task title
</ParamField>

<ParamField body="description" type="string">
  Task description
</ParamField>

<ParamField body="due_date" type="string">
  Due date in ISO 8601 format
</ParamField>

<ParamField body="priority" type="string">
  Priority level: "LOW", "MEDIUM", "HIGH", "URGENT"
</ParamField>

<ParamField body="assigned_to" type="array">
  Array of user IDs assigned to task
</ParamField>

<ParamField body="organizationId" type="string">
  Organization ID for team tasks
</ParamField>

## Example Request

<CodeGroup>
```bash cURL
curl -X POST \
  -H "x-api-key: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Complete documentation",
    "description": "Write API docs",
    "due_date": "2025-01-25T00:00:00.000Z",
    "priority": "HIGH",

  }' \
  https://api.maneged.com/v1/tasks/create
```
